# GradeFlow: PES University GPA Dashboard

A high-performance, mobile-responsive GPA calculator and semester tracker designed specifically for PES University students (RR and EC campuses).

![Version](https://img.shields.io/badge/version-3.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🚀 Features

- **SGPA Calculator**: Supports both direct Grade/Points input and detailed ISA + ESA marks calculation.
- **Semester Tracker**: Log your SGPAs across semesters to visualize your CGPA progression with dynamic charts.
- **What-If Predictor**: Simulate future semester outcomes to see how they impact your final graduation standing.
- **Target Calculator**: Reverse-engineer the SGPA you need next semester to reach a specific target CGPA.
- **Official Reference**: Built-in PES University grading scale, class divisions, and official percentage conversion formulas.
- **Mobile Optimized**: Fully responsive UI with a smooth navigation experience on all devices.

## 🛠️ Built With

- **HTML5/CSS3**: Clean, modern glassmorphic design with a custom dark-mode theme.
- **JavaScript (ES6+)**: Real-time calculation engine with zero dependencies for speed.
- **Chart.js**: Dynamic data visualization for performance tracking.
- **Google Fonts**: Typography powered by DM Sans and DM Mono.

## 📖 How to Use

1. **Calculate SGPA**: Switch between "Grade / Points" or "ISA + ESA Marks" mode. As you type, your SGPA and class status update instantly.
2. **Track CGPA**: Go to the Tracker tab to add semesters. The interactive chart will show your cumulative progress.
3. **Plan Ahead**: Use the Predictor tab to set targets and simulate scenarios.

## ⚖️ Grading System

Based on the official PES University B.Tech grading scale:
- **S Grade**: 90-100 (10 pts)
- **A Grade**: 80-89 (9 pts)
- **B Grade**: 70-79 (8 pts)
- ...and so on.

Percentage Formula: `Percentage = (CGPA − 0.75) × 10`
